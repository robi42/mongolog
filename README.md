# Howto

 1. Install/setup sbt[1].

 2. Build backend code:

  $ cd mongolog
  $ sbt
  $ update
  $ package

 3. Run mongod[2].

 4. Launch Ringo[3] app in `./ringo-app/`.

 5. Happy hacking! :)


  [1]: http://code.google.com/p/simple-build-tool/
  [2]: http://www.mongodb.org/
  [3]: http://ringojs.org/
