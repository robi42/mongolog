# Howto

 1. Get Scala 2.7.7.

 2. Install/setup [sbt].

 3. Build backend code:
    $ cd mongolog
    $ sbt
    $ update
    $ package

 4. Run [mongod].

 5. Launch [Ringo] app in `./ringo-app/`.

 6. Happy hacking! :)


  [sbt]: http://code.google.com/p/simple-build-tool/
  [mongod]: http://www.mongodb.org/
  [Ringo]: http://ringojs.org/
